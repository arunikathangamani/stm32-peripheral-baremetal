# STM32 Peripheral Bare-Metal

A collection of bare-metal STM32 peripheral programming examples developed using Embedded C and direct register-level configuration without HAL.

## About

This repository demonstrates the implementation of STM32 peripherals by directly configuring hardware registers, providing a deeper understanding of the STM32 microcontroller architecture.

## Hardware

- STM32F401RE Nucleo Board

## Programming Language

- Embedded C

## Development Environment

- STM32CubeIDE

## Programming Approach

- Bare-metal Programming
- Register-level Configuration
- No HAL
- No CubeMX-generated peripheral code


# GPIO LED Blink

## Description
This example demonstrates blinking the onboard LED (LD2) connected to PA5 on the STM32F401RE Nucleo board using bare-metal register programming.

## Operation
- Configures GPIOA pin PA5 as an output.
- Continuously toggles the LED with a software delay.

## Hardware
- Board: STM32F401RE Nucleo
- LED: PA5 (LD2)

## Programming Method
- Embedded C
- Bare-metal programming
- Direct register-level configuration
- No HAL or STM32Cube libraries

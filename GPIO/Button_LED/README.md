# Button LED

## Description
This example demonstrates controlling the onboard LED (PA5) using the onboard user button (PC13) on the STM32F401RE Nucleo board.

## Operation
- Press the user button → LED turns ON.
- Release the user button → LED turns OFF.

## Hardware
- Board: STM32F401RE Nucleo
- LED: PA5 (LD2)
- User Button: PC13 (B1)

## Programming Method
- Bare-metal Embedded C
- Direct register-level programming
- No HAL or STM32Cube libraries
